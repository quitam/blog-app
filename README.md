#BLOG APP

Technologies:
+ Front-end: ReactJS
+ Back-end: NodeJS, express
+ Database: MySQL

Web function:
+ Authen (contextAPI): Login, Logout, Register
+ Home: List post, list post by category, create new post (login required)
+ Detail post: Who posted, post date, edit-delete post (my post)
